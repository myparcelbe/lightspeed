Instructions plug-in Lightspeed

Ce manuel vous explique comment réaliser le couplage de Lightspeed avec MyParcel. Il présente également le fonctionnement de ce couplage.

Vous trouverez l’application MyParcel BE dans l’App Store de Lightspeed. Choisissez d’installer l’application. L'application proposée par MyParcel est gratuite. Pour pouvoir l’utiliser, vous devez être client chez MyParcel. Si vous n’êtes pas encore client, vous pouvez vous inscrire sans engagement via le lien suivant : www.sendmyparcel.be/nl-be/registreren.

Sommaire

1 Instructions plug-in Lightspeed

1.1 Configuration

Il est important que vous configuriez d’abord le module MyParcel. Après avoir choisi Installer l’application, vous serez transféré sur la page de configuration suivante:


  1. Introduisez la clé API sous ‘Clé API’. Vous pouvez en faire la demande auprès de MyParcel à l’adresse info@sendmyparcel.be ou sous Paramètres → Général dans MyParcel.
  2. Vous devez ensuite spécifier comment vous souhaitez utiliser le couplage. 

    Vous pouvez vous-même choisir quand vous souhaitez envoyer les données de commande à MyParcel. A cet égard, vous avez deux options: 

    • Orders met status Non envoyé (conseillé)
      Lorsque vous optez pour un transfert des commandes avec le statut ‘Non envoyé’ (conseillé), toutes les commandes avec ce statut sont envoyées directement à MyParcel. Comme toutes les nouvelles commandes dans Lightspeed reçoivent automatiquement le statut ‘Non envoyé’, ce sont toutes vos commandes qui seront donc placées dans MyParcel. Attention. Cela peut prendre jusqu’à 15 minutes. Vous ne devez donc rien faire pour importer vos commandes dans MyParcel.
    • Commandes avec le statut  Envoyé
      Lorsque vous optez pour un transfert des commandes avec le statut ‘Envoyé’, les commandes sont placées dans MyParcel dès que vous spécifiez manuellement dans votre interface Lightspeed que ces commandes sont Envoyées Vous pouvez le faire en cochant les commandes et en choisissant Marquer comme envoyé en haut à droite. Après cela, les commandes seront visibles dans MyParcel dans un délai de 15 minutes. Il se peut que des commandes soient affichées deux fois dans MyParcel si vous avez donné un autre statut à vos commandes et les avez ensuite marquées comme Envoyées alors qu’elles étaient déjà marquées comme telles.

  3. Vous pouvez ensuite indiquer si vous souhaitez générer directement un code Track & Trace dans MyParcel Belgique après importation.

  4. Les envois sont importés en tant qu’envois standards. Si vous le souhaitez, il est possible de les compléter par l’une des options suivantes:  

    • Envoi standard
    • Signature pour réception
    • Signature requise et assurance pour 500 €

1.2 Modification du ou des modes d’expédition

Rendez-vous dans votre interface Lightspeed sur Général → Paramètres → Modes d’expédition. Vous pouvez configurer ici le mode d’expédition par bpost.

Activer l’intégration 
Dans Information de connexion, vous pouvez activer le mode d’expédition. Les autres champs tels que le numéro de client, le code client, le code lieu, l’identifiant et le mot de passe peuvent rester vides. Définissez le ‘Mode’ sur live.


Paramétrer le mode d’expédition 
Vous pouvez indiquer ici les options d'expédition que vous souhaitez afficher sur votre boutique en ligne.

  • Bpack@home: Envoi à l'adresse d’expédition du client
  • Bpack@bpost: Envoi à un point de livraison postale (collecte)


Adresse de l’expéditeur
Introduisez ici l’adresse d’expéditeur de votre boutique en ligne.


Prix
Indiquez ici les prix et le lieu de livraison de votre boutique en ligne.  


1.3 Création des étiquettes

Après avoir donné les bons titres aux modes d’expédition et configuré le module MyParcel selon vos souhaits, vous pouvez commencer à traiter les commandes. Pour ce faire, rendez-vous dans l’interface dans
Commandes.

Si vous avez opté pour le transfert des commandes ayant le statut ‘Non envoyé’, vous ne devrez plus rien faire dans Lightspeed. Lorsque vous recevez une nouvelle commande, celle-ci est automatiquement placée dans MyParcel dans les 15 minutes. Vous pouvez cocher ces commandes dans MyParcel, les traiter et / ou les imprimer.

Si vous avez choisi de transférer les commandes avec le statut ‘Envoyé’, vous pouvez alors cocher, dans votre interface Lightspeed, les commandes que vous souhaitez envoyer via MyParcel.

Après adaptation du statut, la commande est placée dans MyParcel dans les 15 minutes. Vous pouvez cocher cette ou ces commandes dans MyParcel, les traiter et / ou les imprimer.

Attention. Vu que nous ne pouvons pas adresser un nombre trop important de requêtes à Lightspeed, les envois créés sont répercutés auprès de Lightspeed et importés dans MyParcel périodiquement. L’affichage de vos envois dans MyParcel peut donc prendre un certain temps. Après avoir traité les envois dans MyParcel, le code 3S est ajouté à la commande après 2 heures au maximum. Vous ne devez donc plus le faire manuellement.

1.4 E-mail Track & Trace

Si vous choisissez d'envoyer l'e-mail Track & Trace depuis Lightspeed, vous pouvez désactiver cette fonction dans MyParcel sous Paramètres → Track & Trace Si vous préférez envoyer l’e-mail Track & Trace depuis MyParcel, vous pouvez désactiver l’e-mail ‘Order Track & Trace code’. Pour ce faire, laissez le titre et le contenu vides.

1.5 Multishop

Si vous disposez de plusieurs boutiques en ligne liées à un seul compte Ligthspeed, vous devrez les configurer individuellement. Sélectionnez la boutique appropriée et répétez les étapes pour configurer l’application, comme indiqué précédemment sous App store → acheter l’app → installer l’app. Répétez ces étapes pour chaque boutique séparément.

2 Si vous utilisez l’expédition gratuite au-dessus d’un certain montant

Pour utiliser cette option, le module bpost doit être activé. Vous devrez alors paramétrer le module comme suit:

  1. Rendez-vous dans Paramètres → Modes d’expédition → bpost → Prix
  2. Dans Prix, choisissez le bon mode d’expédition ‘Envoi standard bpost’
  3. Activer Enable this pricing.
  4. Introduisez la valeur de la commande pour laquelle vous proposez une livraison gratuite.
  5. Définissez la taxe à 21%.
  6. Revérifiez les prix.
  7. Choisissez Enregistrer.
